# Hydra Task Result

## Summary

Migrated all AI CLI terminals (claude, kimi, gemini, opencode) from clipboard-mediated paste (`"paste"` inputMode) to bracketed paste (`"bracketed-paste"` inputMode). Removed all clipboard dead code since no terminal type uses the paste mode anymore.

## Files Changed

| File | Change |
|------|--------|
| `src/terminal/cliConfig.ts` | Changed inputMode to `"bracketed-paste"` and pasteKeySequence to `() => ""` for claude/kimi/gemini/opencode. Removed `"paste"` from `ComposerInputMode` union. |
| `electron/composer-submit.ts` | Removed `ClipboardSnapshot` type, clipboard deps from `ComposerSubmitDeps`, clipboard params from `createDefaultComposerSubmitDeps`, `snapshotClipboardSafely`, `submitImages`, `createWarning` functions, and the entire paste code path in `submitComposerRequest`. |
| `electron/main.ts` | Removed `clipboard` import from electron, deleted `snapshotClipboard`, `restoreClipboard`, `writeClipboardImage` functions, simplified `createDefaultComposerSubmitDeps` call, removed warning logging block. |
| `tests/composer-submit.test.ts` | Removed `ClipboardSnapshot` import, simplified `createDeps` (no clipboard mocks), replaced clipboard fallback test with two new claude bracketed-paste tests, removed clipboard restore warning test. |
| `tests/terminal-adapters.test.ts` | Updated claude and codex adapter tests to assert `"bracketed-paste"` inputMode and empty pasteKeySequence. |
| `src/types/index.ts` | Removed `capture-clipboard`/`restore-clipboard` from `ComposerSubmitIssueStage`, removed `clipboard-capture-failed`/`clipboard-image-failed`/`clipboard-text-failed`/`clipboard-restore-failed` from `ComposerSubmitIssueCode`, removed warning fields from `ComposerSubmitResult`. |
| `src/components/ComposerBar.tsx` | Removed `formatComposerWarning` function, clipboard stage case branches, and warning handling block. |
| `src/i18n/en.ts` | Removed `composer_stage_capture_clipboard`, `composer_stage_restore_clipboard`, `composer_submit_warning_with_context` keys. |
| `src/i18n/zh.ts` | Removed same i18n keys in Chinese. |
